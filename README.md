# 📱 Unit Converter – Flutter App

A clean, cross‑platform unit converter with **dark mode**, **conversion history**, **swap**, and **copy to clipboard**.  
Built in one day as a portfolio piece to land freelance contracts.

## ✨ Features

- **Length, Weight, Temperature** conversion (metric ↔ imperial)
- **Real‑time calculation** as you type
- **Swap button** to reverse units
- **Copy result** to clipboard
- **Dark / Light mode** (manual or follow system)
- **History** (last 20 conversions, persistent across app restarts)
